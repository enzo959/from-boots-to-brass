# 🪖 FROM BOOTS TO BRASS

## 📌 Description

**FROM BOOTS TO BRASS** est un jeu en ligne de commande développé en **Go**, dans lequel le joueur progresse à travers différents niveaux militaires, en partant de simple recrue ("Boot") jusqu’à atteindre un grade élevé ("Brass").

Ce projet a été réalisé comme une première expérience en Go, avec un focus sur :

* la logique de jeu
* la gestion des entrées utilisateur
* la structuration du code

---

## 🎮 Gameplay

Le joueur évolue dans une simulation où il doit :

* prendre des décisions
* réussir des actions / défis
* progresser en grade

Chaque choix influence la progression et peut mener à différents résultats.

---

## 🛠️ Technologies utilisées

* **Go** (Golang)
* Interface **terminal (CLI)**

## ⚙️ Installation

### 1. Cloner le projet

```bash id="mkyg2v"
git clone https://github.com/enzo959/from-boots-to-brass.git
cd FROM-BOOTS-TO-BRASS
```

### 2. Initialiser Go (si nécessaire)

```bash id="m9vnl8"
go mod init from-boots-to-brass
go mod tidy
```

---

## ▶️ Lancer le jeu

```bash id="t7h3nh"
go run main.go
```
